# README Rewrite Request Triage

Use this after a request arrives.

## Accept when

- the repo is public
- the buyer wants messaging, structure, or CTA help
- the request is centered on one README or one linked product page

## Decline when

- the repo is private
- the buyer expects code changes or implementation
- the work requires deep technical verification or domain credentials
- the buyer wants a full docs overhaul at the base price
