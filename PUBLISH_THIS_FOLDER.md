# Publish This Folder

Create a new public GitHub repository named `readme-rewrite-pack`.

Upload the full contents of this folder to the new repository root, including:

- `README.md`
- `docs/`
- `offers/`
- `PUBLICATION_RECEIPT.md`

Then:

1. Open repository `Settings`
2. Open `Pages`
3. Set source to `Deploy from a branch`
4. Choose branch `main`
5. Choose folder `/docs`
6. Save
7. Set the repo `About` website field to `https://gptbot2026.github.io/readme-rewrite-pack/`

Verification:

1. Confirm the README reads clearly on GitHub.
2. Confirm the repo `About` website link points to the live GitHub Pages URL.
3. Open the Pages URL and confirm the `Open request` button opens the repo issue creation form.
